完了: v0.11.0 PDF Image stamp foundation

foundation plan、JPEG-only pdf_image_stamp core / bridge、PDF Workbench UI、validation、operation plan、実PDF + 実JPEG QAまで完了しました。

  • operation selectorからImage stampを選び、入力PDF・JPEG image・出力PDF、入力PDF summary、pages、position、margin X / Y、width、opacity、rotationを指定できます
  • 全ページまたは指定ページを対象に、center、top left / center / right、bottom left / center / rightのposition presetを利用できます。loading・success・errorも表示します
  • .jpg / .jpegのbaseline RGB / grayscale JPEGに対応しています。PNG alpha / SVG / WebPは未対応です
  • Image stampはadditive visual stylingで、PDF本文編集、redaction、safe redactionではありません。画像を上に置いても下のPDF内容や既存画像・テキスト・ページ番号・watermarkは削除されません
  • stamp-like imagesは電子署名や監査証跡ではありません。protected PDFは復号や権限制限回避を行わず拒否します
  • Inspect / Merge / Split / Extract / Rotate / Delete / Reorder / Text watermark / Page numbers / Image watermark / Text stamp / Text stamp border-backgroundとcompact layoutを維持しています。Delete pagesもredactionではありません
  • PDF preview / rendering / thumbnails、drag positioning、OCR、redaction、safe redaction、direct PDF text editing、既存画像・ページ番号・watermarkの削除や置換、電子署名、監査証跡は未実装です
完了: v0.10.0 PDF text stamp border / background foundation

短いステータスラベルにborder / backgroundを設定し、全ページまたは指定ページへ追加できる基礎機能を公開しました。

  • borderはblack / red / grayと幅・不透明度、backgroundはwhite / yellow / red / grayと不透明度・paddingを設定できます
  • 既存のposition、margin、font size、opacity、rotation、color、operation plan、validationを維持しています
  • Text stampはadditiveで、PDF本文編集やredactionではありません。塗りつぶしやマスクは元の内容を削除せず、安全な墨消しにはなりません
  • APPROVEDやREVIEWED形式のスタンプは電子署名や監査証跡ではありません。保護されたPDFは拒否します
  • PDF preview / rendering / thumbnails、OCR、redaction、direct PDF text editing、image stamp、overlay writing、日本語フォント埋め込みは未対応です
完了: v0.9.1 Text stamp helper layout polish

Text stampカードのStamp text / Pages / Position周辺を読みやすく整え、長いhelper textが3カラム内で詰まって見える問題を改善しました。

  • Text stampのvalidation仕様、pdf_text_stamp、request / response、PDF処理ロジックは変更していません
  • 既存PDF操作を維持し、新しいPDF機能は追加していません
完了: v0.9.0 PDF Text stamp foundation

foundation plan、core / bridge、pdf_text_stamp tool、PDF WorkbenchのText stamp UI、実PDF QAまで完了しました。

  • 短いASCII / Latin-1文字、all pages / selected pages、center、bottom center / right / left、top center / right / leftのposition preset、black / red / grayのcolor presetに対応しました
  • 入力PDF summary、stamp text、pages、position、margin X / Y、font size、opacity、rotation、color、出力PDF、operation plan、validation warning、loading・success・errorを利用できます
  • Inspect / Merge / Split / Extract / Rotate / Delete / Reorder / Text watermark / Page numbers / Image watermark / Text stamp、protected PDF warning、各additive / not redaction warning、compact layout、layout overflow hotfixを維持しています
  • Text stampはadditiveで、PDF本文編集やredactionではなく、既存画像・テキスト・ページ番号・watermarkを削除しません。APPROVEDやREVIEWED形式のスタンプは電子署名や監査証跡ではありません
  • protected PDFは拒否し、復号や権限制限回避は行いません。Delete pagesもredactionではなく、Text / Image watermarkとPage numbersもadditiveでredactionではありません
  • border / backgroundはv0.10.0で追加されました。日本語フォント埋め込み、複数行、任意フォント選択、image stamp UI、overlay writing、実PDF page rendering、preview、thumbnails、OCR、redaction、direct PDF text editingは引き続き未対応です

PDF page numbers foundation
