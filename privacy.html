Privacy notice

Last updated: July 14, 2026

Utility Tools Hub is designed around focused desktop utilities and aims to avoid collecting more information than is reasonably needed. This notice describes the current preview-stage approach and may be updated as the project evolves.

1. Current data practices

The current application does not provide user accounts, paid subscriptions, or an in-app contact form. Core conversion and editing workflows are intended to run locally on the user's computer.

2. Files and project content

Files selected for the app's local tools are processed for the requested desktop operation. Users remain responsible for choosing appropriate files, reviewing outputs, and protecting sensitive content on their devices.

3. Website data

This MVP website does not include analytics, advertising tags, account tracking, or payment processing. A hosting provider may still process basic technical request information under its own policies when serving the site.

4. Downloads and external links

Download buttons lead to GitHub Releases. GitHub may process information under its own terms and privacy practices. External services are not controlled by Utility Tools Hub.

5. Future services

If accounts, payments, support forms, telemetry, or other online services are introduced, this notice is expected to be updated before or alongside those features with more specific information.

6. Security and retention

No system can promise absolute security. Keep local files and project exports in locations you trust, maintain appropriate backups, and avoid placing confidential information in services you have not reviewed.

7. Changes to this notice

This notice may change as the project, website, or applicable requirements develop. The updated date above will be revised when material wording changes.

This is a general project notice for the current MVP and is not intended as legal advice.
