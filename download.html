v0.4.4の更新内容

軽量なOperation plan previewの読みやすさと警告表示を改善したQA polishリリースです。

  • ページ指定chipsとselected pagesの読みやすさを改善
  • 不正なページ指定と範囲外ページ指定のwarningを改善
  • 全ページ削除になりそうな場合のリスク警告を改善
  • Merge / Split / Extract / Rotate / Deleteの操作予定表示を整理
  • protected PDF warningと、Delete pagesはredaction（墨消し）ではない注意を継続
  • Inspect / Merge / Split / Extract / Rotate / Deleteは引き続き利用可能
  • PDF処理はローカル実行
  • これは本物のPDFページ描画ではなく、実PDF rendering、thumbnails、reorderは未実装
  • OCR / redaction / direct PDF text editing、復号・権限制限回避・パスワード処理は未実装

Windowsの警告表示について

個人開発のアプリのため、インストール時に「認識されていないアプリ」などの警告が表示される場合があります。上記の公式リンクから入手したファイルか確認し、内容に納得したうえで操作してください。
