v0.7.1の更新内容

Page numbers real PDF QA polishとして、既存機能の実PDF確認観点を整理しました。新しいPDF処理機能の追加ではありません。

  • all pages / selected pagesの出力確認観点を整理しました
  • start number / format / positionの代表ケースを確認観点に追加しました
  • 出力PDFが開けること、ページ数が入力PDFと同じことを確認観点に追加しました
  • compact layoutとlayout overflow hotfixを維持しています
  • Page numbersはadditiveであり、PDF本文編集やredaction（墨消し）ではありません
  • 既存ページ番号を検出・削除・置換する機能ではありません
  • Inspect / Merge / Split / Extract / Rotate / Delete / Reorder / Text watermark / Page numbersは引き続き利用可能です
  • protected PDF warning、Delete pagesはredactionではない注意、WatermarkとPage numbersはadditiveである注意を維持しています
  • PDF処理ロジック、tool_id、request/response形式は変更していません
  • PDF処理はローカルで実行されます。暗号化PDFの復号、権限制限回避、パスワード処理には対応していません
  • 実PDF rendering、preview、thumbnails、OCR、redaction、direct PDF text editingは未実装です
