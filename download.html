v0.3.3の更新内容

PDF tools QA and UX polishとして、既存PDF操作の案内と状態表示を改善しました。

  • PDF Toolsの説明、入力例、disabled理由、エラー表示を改善
  • loading / success / errorの状態を分かりやすく整理
  • Merge / Split / Extract / Rotate / Deleteは引き続き利用可能
  • Delete pagesはページ単位の削除であり、墨消しではありません
  • PDF処理はローカル実行され、Python sidecarは不使用
  • 新しいPDF処理機能は追加していません
  • OCR・墨消し・PDF本文編集は未実装
