v0.10.0の更新内容

PDF text stamp border / background foundationとして、短いステータスラベルの視覚スタイルを調整できるようになりました。

  • borderはblack / red / grayと、幅・不透明度を設定できます
  • backgroundはwhite / yellow / red / grayと、不透明度・paddingを設定できます
  • 全ページまたは指定ページを対象にでき、既存の位置、X/Y余白、font size、不透明度、rotation、文字色も引き続き利用できます
  • 実行前にoperation planとvalidationを確認できます。保護されたPDFは拒否します
  • Text stampはadditiveな視覚表現で、PDF本文編集やredactionではありません。塗りつぶしやマスクで覆っても元の内容は削除されず、安全な墨消しにはなりません
  • 既存の画像・テキスト・ページ番号・watermarkは削除されません。APPROVEDやREVIEWED形式のスタンプは電子署名や監査証跡ではありません
  • 既存のPDF Workbench操作は引き続き利用できます。PDF preview / rendering / thumbnails、OCR、redaction、PDF本文編集、image stamp、overlay writing、日本語フォント埋め込みは未対応です
  • ダウンロードは公式GitHub Releaseから行ってください
