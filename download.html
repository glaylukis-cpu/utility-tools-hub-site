v0.6.2の更新内容

PDF選択後にPDF Workbenchのcompact layoutが崩れる問題を修正したhotfixです。新しいPDF処理機能は追加していません。

  • 長いPDFファイル名がカードやサイドバーを押し広げないよう改善
  • 入力PDF summary gridと操作カードを親カード内に収めるよう調整
  • operation selectorと選択中の操作カードを中央カラム内に保持
  • 左のactive-selection panelと右のstatus / safety panelの幅を安定化
  • 1280px幅や100% / 125%表示で横方向に崩れにくいよう改善
  • 一度に1つの操作を表示するcompact layoutの方針を維持
  • Inspect / Merge / Split / Extract / Rotate / Delete / Reorder / Text watermarkは引き続き利用可能
  • PDF処理はローカル実行。処理ロジック、tool_id、request / response形式に変更なし
  • Watermarkはadditiveであり、PDF本文編集やredaction（墨消し）ではありません
  • Delete pagesはページ単位の削除であり、redactionではありません
  • 実PDF preview / rendering / thumbnailsは未実装
  • OCR / redaction / direct PDF text editing、復号・権限制限回避・パスワード処理は未実装
